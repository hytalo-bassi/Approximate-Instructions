# Changelog

All notable changes to this project will be documented in this file.

## [v0.0.1] - 2026-07-08

### Added
- Initial public release for the Approximate-Instructions project.
- Support for approximate integer operations (`addx`, `subx`, `mulx`, `divx`).
- Support for approximate floating-point operations (`faddx`, `fsubx`, `fmulx`, `fdivx`).
- Patch sets for riscv-opcodes, binutils, gdb, and gem5 integration.
- Docker environments for base toolchain, gem5 base, and gem5 full workflows.
- Documentation for installation, development workflow, bitfield encoding, and examples.

### Notes
- This release establishes the baseline structure and workflow for future versioned releases.
